家計簿

シンプルに、賢く。毎日の家計を記録するアプリ。

アプリについて

「家計簿」は、収支の記録・カテゴリ分析・レシートOCR・家族グループ共有などの機能を備えたスマートフォン向け家計管理アプリです。

  • 収入・支出の記録と月別グラフ表示
  • レシートをカメラで撮影して自動入力（AI OCR）
  • 家族グループで収支をリアルタイム共有
  • カレンダーで日別収支を一覧確認
